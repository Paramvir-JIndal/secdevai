# Contributing to SecDevAI

Contributions to the project are always welcome. Please feel free to create pull requests, or report issues. 

> **Note:** For guidance on adding new security context such as rules, tools, or languages, see the **"Extending SecDevAI"** section in [`docs/usage.md`](docs/usage.md#extending-secdevai).

Once you have verified that your context extension works well, you are encouraged to contribute back by creating a pull request to update the templates. AI-generated code is welcome, but please ensure you have tested it and provide clear descriptions of the changes in your PR.
